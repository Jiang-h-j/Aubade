# Aubade 原型 Demo

这是 Aubade 记账 App 的**可点击低保真原型**，用于产品评审，不是最终 App，也不是生产代码。真实 App 将用 SwiftUI 实现。

## 怎么打开

**双击 `app/index.html`**，用浏览器（Chrome / Safari / Edge 均可）打开即可，无需安装任何环境。

页面中间是一个模拟的 iPhone 屏幕，直接用鼠标点。屏幕下方有一个"原型演示控制台"（不属于 App 本身），用来：

- **重置数据**：清空本次演示数据，回到"首次使用"引导。
- **📷 模拟快捷指令截图**：假装你在付款页触发了 iOS 快捷指令把截图发给 App，体验"后台识别 → 直接入账 → 顶部通知"的主路径（不进 App 界面、不打断当前页）。
- **模拟识别失败**：勾选后再去记账（含快捷指令截图），体验"没识别出金额 → 转手动 / 失败通知"的异常流程。
- **模拟未配置 Key**：勾选后再用截图/语音记账，体验"提示先配置 DeepSeek Key"的拦截。

> 注意：这是原型，数据不落盘，**刷新页面会回到默认状态**。

## 建议体验路径

对应设计文档《主要评审路径》一节，简版：

1. 打开默认就在「记账」页。
2. **体验主打的快捷指令截图**：点「截图识别」看说明卡 → 点「演示：模拟收到一张快捷指令截图」→ 看顶部通知"后台识别中→已记一笔 ¥42.80"，记账页不被打断、最近记录立刻出现这笔 → 点通知打开结果卡片改一下。（也可用底部控制台的「📷 模拟快捷指令截图」直接触发。）
3. 点「文本识别」→ "读取剪贴板"填入示例 → "识别并记账" → 看识别中 → 看结果卡片 → 当场改分类 → 完成。
4. 切到「账单」页看新增的几笔和剩余总额变化。
5. 进「统计」用顶部「日/周/月/年」切粒度、「‹ ›」翻时间：切「日」选某天看当天流水，切「月」翻上月，切「年」看全年每月趋势；点分类占比里的某一类可查看该分类的记录明细，看分类占比和月预算超支标红。
6. 进「我的」看剩余总额、调整初始总额、预算、DeepSeek Key、分类。
7. 用底部演示控制台试异常态（勾"模拟识别失败"或"模拟未配置 Key"后触发快捷指令截图），或"重置数据"体验首次引导。
